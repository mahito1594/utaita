# ユーザーストーリー (正本)

> この日本語版が正本。フェーズの区切りで英語版 [stories.md](./stories.md) に同期する。

形式: 1 ストーリー 1 行。自明でない場合のみ受け入れ条件を添える。フェーズごとにグループ化し、未割り当てのアイデアは Icebox へ。

> 状態: ロードマップから起こした初期ドラフト。pleroma-fe への実際の不満を観察して書き直していくこと。

## Phase 1 — 閲覧 MVP

- [ ] インスタンスのアカウントでログインでき、ブラウザを再起動してもログイン状態が保持される。
- [ ] Home タイムラインを滑らかな無限スクロールで読める。CW を開いてもスクロール位置がずれない。
- [ ] home / local / bubble / federated を切り替えられ、セッション中は各タイムラインのスクロール位置が保持される。
- [ ] 誰がブーストしたか、どんな絵文字リアクションが付いているかが見える。カスタム絵文字はインラインで描画される。
- [ ] 投稿を開いて会話ツリー全体を読める。どの投稿から来たのか見失わない。
- [ ] プロフィールを投稿 / 返信 / メディアのタブ付きで閲覧できる。
- [ ] 通知を読める。メンション・ブースト・フォロー・絵文字リアクションが一目で区別できる。
- [ ] スマホ (片手での操作範囲、横スクロールなし) と PC の両方で快適に使える。
- [ ] ライト / ダークテーマを切り替えられる。

## Phase 2 — 書き込み

- [ ] CW・公開範囲 (`local` 含む) 付きで投稿でき、カスタム絵文字の補完が効く。
- [ ] alt text 付きで画像を添付でき、アップロードの進捗が見える。
- [ ] タイムライン上からファボ・ブースト・ブックマーク・絵文字リアクションができる。
- [ ] プロフィールからフォロー / 解除でき、承認制アカウントには申請中の状態が表示される。
- [ ] アカウント・ハッシュタグ・投稿を検索できる。

## Icebox

- 見逃した投稿のキャッチアップダイジェスト (Phanpy 風)
- タブを閉じても消えない下書き
- Streaming によるタイムラインのリアルタイム更新
- 複数アカウント
