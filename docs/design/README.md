# design

- Design token decisions (palette, typography scale, spacing, radius, dark
  mode) and their rationale. The tokens themselves live in `panda.config.ts`;
  this directory records why they are what they are.
- Wireframe exports (Excalidraw or similar), named `<screen>-<yyyymmdd>.*`.

Working agreement: tokens first, components extracted later. A pattern is
promoted to a Panda recipe only after it has appeared three times in real
screens — an instance of the rule of three in
[process.md](../process.md#working-agreements).
